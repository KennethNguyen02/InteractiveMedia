# Interactive Media

This repository contains my work from the **Interactive Media** course at **Konkuk University**.

The course introduces the basics of **interactive digital media and creative coding**, where we explore how programming can be used to create interactive visual and sound-based artworks.

Throughout the semester we work with tools such as **p5.js**, **Java**, and **Python** to experiment with:

- Data types and variables  
- Conditional logic and loops  
- Arrays and object-oriented programming  
- Image and webcam interaction  
- Sound and audiovisual interaction  

The course includes weekly exercises, a **midterm project**, and a **final interactive media project**.
